# User Management App

React-приложение для управления пользователями с авторизацией.

## Технологии

- TypeScript
- React 16
- Webpack + Npm
- React Router 6
- Ant Design 5
- styled-components
- TanStack Query 4
- axios
- dayjs
- FSD архитектура

## Установка

```bash
npm install
```

## Запуск

```bash
npm run start
```

Приложение откроется на http://localhost:3000

## Структура проекта (FSD)

```
src/
├── app/           # Инициализация приложения
├── pages/         # Страницы
├── widgets/       # Крупные блоки
├── features/      # Бизнес-функции
├── entities/      # Сущности
└── shared/        # Переиспользуемый код
```
